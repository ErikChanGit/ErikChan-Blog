#### 部署项目

> 部署到 Tomcat

- 下载 apache-tomcat-9.0.50， 解压到指定目录

- 将项目放到 webapps 目录下

- 启动Tomcat

  ```
  ./bin/startup/sh
  ```

> 部署到 njix



> 使用 redis

- 安装 redis

  ```
  wget http://download.redis.io/releases/redis-6.0.8.tar.gz
  tar xzf redis-6.0.8.tar.gz
  ```

- 编译

  ```
  cd redis-6.0.8
  make
  ```

- 启动服务

  ```
  ./redis-server ../redis.conf
  ```
